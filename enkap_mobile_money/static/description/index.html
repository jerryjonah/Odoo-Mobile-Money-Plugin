📋 Quick Installation

  1. Install the addon: Go to Apps → Search "SmobilPay" → Install
  2. Configure API: Settings → Payment Providers → SmobilPay → Configure
  3. Enter credentials: Add your Consumer Key and Consumer Secret
  4. Test connection: Use the built-in connection test
  5. Go live: Switch to production mode when ready
